# Grants AI

AI-powered grant discovery and application assistant from Aurora Market.

## Features
- Intelligent grant matching engine
- Streamlined application process
- Comprehensive funding database
- AI-powered recommendations

## Deployment & Environment Requirements
- Deployment target: Aurora Market platform
- CI workflow: `.github/workflows/ci.yml` validates required files and merge-marker safety checks
- Environment variables are listed in `.env.example` for backend/payment integrations
